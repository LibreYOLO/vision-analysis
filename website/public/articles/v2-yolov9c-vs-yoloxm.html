YOLOv9-C vs YOLOX-M: Benchmark Comparison on NVIDIA A100

COCO val2017 · 5,000 images · PyTorch FP32 · March 2026 · Benchmarked with LibreYOLO v0.1.2

A heavyweight matchup between two similarly-sized models from different eras. YOLOv9-C achieves 44.6 mAP@50-95 at 35.9 FPS, while YOLOX-M achieves 41.5 mAP@50-95 at 43.1 FPS. YOLOv9-C (Academia Sinica, ECCV 2024) wins on accuracy by 3.1 mAP points; YOLOX-M (Megvii, 2021) wins on speed by 20%. At nearly identical parameter counts (~25M), this comparison isolates pure architectural innovation — YOLOv9's PGI and GELAN vs YOLOX's decoupled head and SimOTA. Both benchmarked with LibreYOLO v0.1.2, retrained weights.

Frequently Asked Questions

Which is better, YOLOv9-C or YOLOX-M?
YOLOv9-C (ECCV 2024) wins on accuracy: 44.6 vs 41.5 mAP@50-95. YOLOX-M (Megvii, 2021) wins on speed: 43.1 vs 35.9 FPS. At identical parameter counts (~25M), YOLOv9-C trades 20% speed for 7.5% higher accuracy. Both available in LibreYOLO.
Is this a fair comparison?
Yes. Both models have nearly identical parameter counts (25.5M vs 25.3M), both are anchor-free with NMS post-processing, both trained on COCO train2017, and both benchmarked with LibreYOLO v0.1.2 using retrained weights on the same A100 hardware.
How can I reproduce these results?
Install LibreYOLO (pip install libreyolo), then run LibreYOLO("yolov9-c").val(data="coco.yaml") or LibreYOLO("yolox-m").val(data="coco.yaml"). All results from LibreYOLO v0.1.2, COCO val2017, 5,000 images.
