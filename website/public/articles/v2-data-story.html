YOLOv9-T vs YOLOX-S on NVIDIA A100

COCO val2017 · 5,000 images · PyTorch FP32 · March 2026

The story in 10 seconds
YOLOX-S is faster (44 FPS vs 23 FPS) and more accurate (34.6 vs 30.2 mAP). But YOLOv9-T punches far above its weight — with only 2.04M parameters (vs 8.97M), it delivers 3.8x more mAP per parameter. If you're deploying to edge hardware where every megabyte counts, YOLOv9-T is the smarter pick. For cloud inference where raw accuracy matters, go YOLOX-S.

When to Use Each Model

Choose YOLOv9-T when...

  • Deploying to edge devices (Jetson, RPi)
  • Model size budget under 5M params
  • Maximizing accuracy-per-FLOP
  • Memory-constrained environments

Choose YOLOX-S when...

  • Running on cloud GPUs (A100, V100)
  • Raw accuracy is the priority
  • High recall is critical (find everything)
  • Latency budget is tight (<25ms)
